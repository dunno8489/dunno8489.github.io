PS3 Error Code: 80030002

Meaning:

Your HDD/SSD is dying or the File System is
dying.

Solutions:

1. Restart The Console.
2. Check For System Updates.
3. Rebuild Database.
4. Check the HDD/SSD.
5. Restore File System.

IF NONE ELSE WORKS:

1. Contact PlayStation™ Support.
2. Send Your PlayStation™3 System to Repair.
